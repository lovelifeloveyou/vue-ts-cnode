# vue-ts-cnode
🚀 基于vue+typescript构建的仿[CNode社区](https://cnodejs.org/)

## 项目安装及启动
```
npm install
npm run serve
```

## API清单 x 15
- [√] 主题首页
- [√] 主题详情
- [√] 新建主题
- [√] 编辑主题
- [√] 收藏主题
- [√] 取消收藏主题
- [√] 用户所收藏的主题
- [√] 新建评论
- [√] 为评论点赞
- [√] 用户详情/个人主页
- [√] 登录
- [√] 获取未读消息数
- [√] 获取已读和未读消息
- [√] 标记全部已读
- [√] 标记单个消息为已读

## 技术栈
- Vue @2.6.11
- TypeScript
- TSX
- LESS
